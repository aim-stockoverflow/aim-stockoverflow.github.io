#### 安装npm所需模块

```bash
$ npm install

# 如果安装出错，可以安装淘宝提供的镜像cnpm(需安装cnpm)
$ cnpm install

# 或更换国内镜像
$ npm config set registry http://registry.cnpmjs.org
```

#### 运行开发环境

```bash
$ npm run dev
```

#### 打包静态资源

```bash
$ npm run build
```
